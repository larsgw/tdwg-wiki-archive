---+++ How - survey methods / intensity / protocol

   * Protocol Name - a formal or informal name for the protocol (can be &#8220;none&#8221; or &#8220;unknown&#8221;)
   * Protocol Description - text description of the protocol / methods used
   * Protocol Type - the type of protocol
   * Protocol Reference &#8211; citation for a published protocol (can include links to online resources or protocol library)
   * Search Adequacy &#8211; comments on search thoroughness, habitat, methods, environment, observer experience, ease of detection, timing. 
   * Additional Inventory Needed - Is additional inventory of the area needed for this element? 
   * Level of Effort - effort expended on a given visit
