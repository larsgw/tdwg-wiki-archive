The following is a fully expanded xml schema diagram showing all elements of MODS (Version 3.0 December 5, 2003). Many elements carry rich attributes; these are listed in the annotations after [ATTR:]. At the end you can find the schema as a zip file. It has been modified by adding the attribute annotations mentioned above and by fixing some validation problems in the original version as downloaded from http://www.loc.gov/standards/mods/; see comments at the start of the schema.
