Restricted to integer values from 0 to 5. 0 is defined as unspecified level, and 1 to 5 indicates expertise from schoolchildren to taxonomic expert. See the description of the values for recommendations for interpreting and choosing the expert level.

See [[http://wiki.cs.umb.edu/twiki/pub/UBIF/EnumeratedValues/UBIF-Docu-Enumerations.html#ExpertiseLevelEnum][UBIF enumeration info]] for a tabular overview of values.
