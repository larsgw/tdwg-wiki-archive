This is a place-holder for information about the Species microformat - a protocol for marking up taxonomic names (in all ranks) using HTML classes, so that they become discoverable and parsable by user agents such as the [[http://www.kaply.com/weblog/2007/06/04/operator-08-is-available/][Operator extension]] for Firefox.

See
   * http://microformats.org/wiki/species - for work in progress (technical)
   * http://species.wikimedia.org/wiki/Wikispecies:Microformat - for a primer and further explanation.
