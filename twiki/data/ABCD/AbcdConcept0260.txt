<verbatim>DataSets/DataSet/Units/Unit/UnitContentContacts/UnitContentContact/Addresses/Address</verbatim>

[@preferred]

---+++++ Group

Agent

---+++++ Subgroup


---+++++ Full Name

Preferred

---+++++ Documentation

A flag to set to indicate the preferred address for the administrative contact for the unit record. 

---+++++ Content

The address marked as preferred by means of this attribute should be the first approached. The other addresses named should be regarded as alternate addresses. 

---++++ Recommended or Prescribed Values

true, false, or empty. For a single unbounded element there should only be one with the @preferred attribute set to true. 

---+++++ Example Value

true

---+++++ Example Explanation

The address with its attribute set to true is the preferred address.
