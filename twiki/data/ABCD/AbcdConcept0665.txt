---+++++ Identifier

<verbatim>DataSets/DataSet/Units/Unit/ObservationUnit/ObservationUnitIdentifiers/ObservationUnitIdentifier</verbatim>

[@preferred]

---+++++ Group

Meta

---+++++ Subgroup

Identifiers

---+++++ Full Name

Preferred

---+++++ Documentation

A flag set to indicate the preferred observation unit identifer.

---+++++ Content

The identifier marked as preferred by means of this attribute should be the first used. The other identifiers named should be regarded as alternate identifiers.

---++++ Recommended or Prescribed Values

true, false or empty. For a single unbounded element there should only be one with the @preferred attribute set to true.

---+++++ Example Value

true

---+++++ Example Explanation

The element with its attribute set to true is the preferred identifer.
