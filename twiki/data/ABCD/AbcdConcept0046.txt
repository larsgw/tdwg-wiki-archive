---+++++ Full Name

Data source last modification date

---+++++ Documentation

The last modification date for the data source.

---+++++ Content

The last modification date for the data source. Source for the Dublin-Core element Date.Modified. If the provider cannot assess this for online data sources, the current date/time may be substituted. For legacy data this may be set to the file date of imported data, or estimated. Note that it may well be that the result of the query, i.e. the document containing this datum, does not contain the parts of the souce that have actually been changed.
