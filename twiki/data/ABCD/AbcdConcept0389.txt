---+++++ Full Name

End of unit identification (Julian day number)

---+++++ Documentation

Note that this element is not normally used in the context of an identification event.

The Julian day number for the last day in the unit identification event period.

---+++++ Content

The Julian day number (sequential day within the year) of the last day in the unit identification event period.

The year is given in <verbatim>ISODateTimeEnd</verbatim>.

The content range is 1-365 for normal years, 366 for leap years.
