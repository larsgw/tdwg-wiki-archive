[@Preferred]

---+++++ Group

Agent

---+++++ Subgroup


---+++++ Full Name

Preferred

---+++++ Documentation

A flag set to indicate the preferred URL for the specimen owner.

---+++++ Content

The URL marked as preferred by means of this attribute should be the first used. The other URLs named should be regarded as alternative URLs.

---++++ Recommended or Prescribed Values

true, false, or empty. For a single unbounded element there should only be one with the @preferred attribute set to true.

---+++++ Example Value

true

---+++++ Example Explanation

The element with its attribute set to true is the preferred URL.
