Container element for several language-specific representations of the full organisation or corporate name for the data source owner.

---+++++ Content

Container element for several language-specific representations of the full organisation or corporate name for the data source owner (en: 'Botanical Garden of ...', de: 'Botanischer Garten von ...'). (vCard: Org.Org-Name). The Label mechanism also supports acronyms/abbreviations (no vCard equivalent!).
