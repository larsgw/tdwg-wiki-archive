Gathering project contact (inherited name prefix)

---+++++ Documentation

The prefix to the inherited name of the person to contact for information about the gathering project.

---+++++ Content

The prefix to the inherited name of the person to contact for information about the gathering project. The prefix should be output before the name, but is usually not included in sorting. Compare Title for 'Prof', 'Dr.' (vCard: N-Prefix)
