[@preferred]

---+++++ Group
Meta

---+++++ Subgroup
UDDI

---+++++ Full Name
Preferred 

---+++++ Documentation

A flag set to indicate the preferred technial contact. 

---+++++ Content

The contact marked as preferred by means of this attribute should be the first approached, the other contacts named should be regarded as alternate contacts. 

---++++ Recommended or Prescribed Values

true, false or empty

For a single unbounded element there should only be one with the @preferred attribute set to true.

---+++++ Example Value

true

---+++++ Example Explanation

The element with its attribute set to true is the preferred contact.
