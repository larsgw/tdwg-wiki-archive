---+++++ Full Name

Gathering event period is explicit

---+++++ Documentation

This is an indicator for the expliciteness of the gathering event time period.

---+++++ Content

This is an indicator for the expliciteness of the gathering event time period. 

This is set to 'True' if the gathering event takes place for the period indicated. It is set to false if it occurs at some point of time during the period, and is not filled if unknown or if no period is specified.

---++++ Recommended or Prescribed Values

true 

false 

empty

---+++++ Example Value

The execution of the unit identification event took exactly the time given as the period. 

The unit identification event took place at some point during the period recorded here. 

The exact timing and/or duration of the identification event is unknown or no period is specified.
