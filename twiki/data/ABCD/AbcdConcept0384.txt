The time zone to which the time-of-day data for the unit identification event refer.

---+++++ Content

The time zone to which the time-of-day data for the unit identification event refer. 

Timezone expressed in minutes in accordance with the "seven property model" for date/time attributes in XML schema (http://www.w3.org/TR/2004/WD-xmlschema11-2-20040716/#theSevenPropertyModel).
