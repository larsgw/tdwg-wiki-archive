[@preferred]

---+++++ Group

Agent

---+++++ Subgroup


---+++++ Full Name

Preferred

---+++++ Documentation

A flag set to indicate the preferred email address for the specimen owner. 

---+++++ Content

The email address marked as preferred by means of this attribute should be the first approached. The other email addresses named should be regarded as alternative email addresses.

---++++ Recommended or Prescribed Values

true, false, or empty. For a single unbounded element there should only be one with the @preferred attribute set to true.

---+++++ Example Value

true

---+++++ Example Explanation

The element with its attribute set to true is the preferred email address.
