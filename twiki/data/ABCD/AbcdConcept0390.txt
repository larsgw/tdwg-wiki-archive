---+++++ Full Name

End of unit identification (time of day)

---+++++ Documentation

The time of day in hours from midnight for the end of the unit identification event period.

---+++++ Content
Note that this element is not normally used in the context of an identification event.

The time of day in decimal hours from midnight local time for the end of the unit identification event period.

Note that times are decimal hours: 13.5 represents 1:30 pm, not 1:05 pm, nor 1:50 pm.

---++++ Recommended or Prescribed Values


---+++++ Example Value

13.5

04.25
