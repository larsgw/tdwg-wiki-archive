---+++++ Documentation

A container for the datasource description in a specified language.

---+++++ Content

This is a container for the datasource description (title, description, etc.) in a specified language. The attribute @language lets you specify the language, the Representation mechanism makes it possible to ensure that only one name per language is specified.
