---++Visualization Tools

Visualization tools will read the Bioblitz datastore and present the data in an interested way. Because the data will be stored in a public Fusion Table, any developer that want to do visualization has it very easy.

Here is a [[http://wiki.tdwg.org/twiki/bin/view/Main/BioBlitzTallymaster/][link to Tallymaster]] page with some requirements for visualization tools for developers to get ideas.
