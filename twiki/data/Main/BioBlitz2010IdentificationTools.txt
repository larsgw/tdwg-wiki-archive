Identification tools will be helping on the process of identifying images. When users of Data gathering tools can not identify their observations,they can flag as "IdentificationRequiered". The tools here will read the Bioblitz data store and present it to interested taxonomist. Those taxonomist will provide identifications so that observations will be completed.

We will also use this page to describe the whole workflow.
