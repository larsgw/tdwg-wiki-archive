---++ PDF Rendering Variables
The following variables control the generation of the document cover page:
   * Set GENPDFADDON_BANNER = Taxonomic Databases Working Group (TDWG)
   * Set GENPDFADDON_TITLE = TDWG Constitution
   * Set GENPDFADDON_SUBTITLE = By The Process Subgroup
   * Set GENPDFADDON_TITLETOPIC = Sandbox.PdfGenerationTestFrontpage
   * Set GENPDFADDON_MARGINS = top:0.5in,bottom:0.5in,left:0.8in,right:1.0in
   * Set PDFORIENTATION = landscape
