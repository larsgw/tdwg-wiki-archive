*Description:* tapirChirp is a TAPIR client library written in PHP. It can be used to interact with a single provider endpoint through methods that correspond to each TAPIR operation. It supports all request encodings. tapirChirp was originally developed to be used by the TapirTester.

*Download:* http://sourceforge.net/project/showfiles.php?group_id=38190&package_id=256783

*Subversion repository:* https://digir.svn.sourceforge.net/svnroot/digir/tapirchirp

*Language:* PHP (compatible only with PHP5).
