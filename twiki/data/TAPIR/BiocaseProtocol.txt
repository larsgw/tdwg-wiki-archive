---+++ Improved BioCASe protocol version 2
V2 was never released, but the second version existed as an xml schema and had thoughts about adding :
   * a metadata request type
   * allows to specify parts of an existing schema to be returned only ("partial search")
   * allows to freely specify the structure of returned data ("flexible search")
