*Description:* TapirJChirp is a TAPIR client library written in Java (ported from the tapirChirp PHP library). It can be used to interact with a single provider through methods that correspond to each TAPIR operation and it supports all request encodings.

*Subversion repository:* https://digir.svn.sourceforge.net/svnroot/digir/TapirJChirp

*Language:* Java

*Developer:* JamesYtow
