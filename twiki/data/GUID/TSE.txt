---+++ Taxonomic Search Engine

The [[http://darwin.zoology.gla.ac.uk/~rpage/portal][Taxonomic Search Engine]] provides LSIDs for the taxonomic databases that it queries, namely ITIS, IPNI, Index Fungorum, NCBI, uBio, Hymenoptera Name Server, and Sp2000. The search engine itself is described in a paper in [[http://www.biomedcentral.com/1471-2105/6/48][BMC Bioinformatics]], and source code is available on [[http://taxonomicsearch.sourceforge.net/][SourceForge]].
